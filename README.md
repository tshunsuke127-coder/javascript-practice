JavaScript 一覧表示アプリ（クラス設計）

## 概要
一覧表示アプリの状態管理をクラスで設計し直し、
Store（状態管理）とUI（表示・操作）を分離しました。

## 使用技術
- HTML
- CSS
- JavaScript（Vanilla JS）
- localStorage

## 機能一覧
- CRUD操作
- 状態管理（Storeクラス）
- localStorage 永続化
- UIとロジックの分離

## 工夫した点
- state管理をStoreクラスに集約しました
- UIはDOM操作のみに責務を限定しました
- 変更に強い設計を意識しました

## 学んだこと
- クラス設計による責務分離
- 状態管理の重要性
- 保守しやすい構造の作り方
